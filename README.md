# jquery.CoinBarChart
A lightweight jquery plugin for a basic bar chart cotaining 3d coins

## Default Options
- labels:[], `if not provided, series data will be used as labels`
- series:[],
- strokeWidth:60,
- topLabel:true,
- bottomLabel:false,

## Example
```javascript
	$('#barchart').barchart({
		labels:['Sales','Services'],
		series:[1000, 2000],
		coins:15,
		fadeIn:50
	});
```
